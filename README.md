# Kane Pubs

Contacts · Organizations · Business

## Stack

- Next.js 14 (App Router)
- React 18
- TypeScript

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy (Vercel)

Connect this repo to Vercel. Add environment variables in the Vercel dashboard (e.g. `DATABASE_URL`, `SMTP_*`) — do not commit `.env`.

- **New Contact** — `/new-contact`
